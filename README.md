MUDÈRA // LUXURY E-COMMERCE
MUDÈRA is a high-end, minimalist e-commerce platform designed for luxury streetwear. It features a custom-built, lightweight cart system, dynamic product variants, and a smooth, cinematic user experience powered by GSAP.

Features
Persistent Shopping Atelier: Uses localStorage to keep the user's cart saved across all pages.

Dynamic Product Swapping: Real-time image and color variant updates.

Interactive UI: Smooth drawer animations for the "Atelier" (cart) using GSAP.

Order Validation: Secure client-side checkout validation.

UX Enhancements: Built-in quantity selection, responsive layout, and "Back-to-Top" utility.

Tech Stack
HTML5

CSS3 (with Modern Flexbox/Grid)

Vanilla JavaScript (ES6+)

GSAP (GreenSock Animation Platform)

Project Structure
/index.html - Landing page.

/product-page.html - Product details and configuration.

/checkout.html - Secure checkout form.

/scripts.js - Global state management and animation logic.

/img/ - Product imagery and branding assets.

How to Run
Clone this repository to your local machine.

Open index.html in any modern web browser.

Ensure your browser supports localStorage to enable the "Atelier" functionality.
